# Projet

Projet C Automne 2025/2026 pour l'université

## Résumé

Il s'agit de simuler un **gestionnaire de fichiers** en C

## Utilisation

### Prérequis

1. Avoir `gcc` installé et fonctionnel afin de compiler du C
2. Avoir `make` permettant de compiler automatiquement le projet

⚠️
> Les exécutables produits peuvent potentiellement ne pas fonctionner correctement sur d'autres plateformes que linux

### Procédé

1. Clonez le code `git clone https://github.com/Greensky-gs/projet-c-automone-2025`
2. Lancez la commande (sur le dossier cloné) : `make -f makefileNUMERO`, en remplaçant **NUMERO** par `1`, `2`, `3` ou `4` pour avoir l'une des quatres versions demandées du projet (en cours de développement)
